Application

The Disaster Web Management System is a robust platform designed to facilitate effective communication and action during emergencies. Built using HTML, JavaScript, C#, and CSS, this system serves as a vital tool for both users and administrators in managing disaster-related information and resources. Key Features: 1. User Functionality: Report Disasters: Users can easily report new disasters, providing essential information such as location, type of disaster, and any relevant details. This feature enables timely updates and alerts to the community. View Active Disasters: A dedicated section allows users to view a list of currently active disasters. This includes details like the nature of the disaster, affected areas, and real-time updates, helping users stay informed. Donation Options: Users can choose to donate either goods or money directly through the platform. This feature includes options for specifying the type of goods they wish to donate, ensuring that contributions meet the needs of those affected. Track Donations: A user-friendly dashboard enables donors to track their contributions. Users can see a history of their donations, including amounts and types of goods donated, fostering transparency and trust. 2. Administrator Functionality: View Potential Donors: Administrators have access to a comprehensive list of potential donors, allowing them to manage and communicate with individuals who have shown interest in contributing. This feature supports targeted outreach for specific needs during disasters. Manage Reports: Admins can review and validate reported disasters, ensuring that accurate and relevant information is disseminated to the public. Resource Allocation: The system allows administrators to manage and allocate resources efficiently based on reported needs and available donations. 3.Technical Overview: HTML & CSS: For structuring and styling the user interface, ensuring an intuitive and accessible design. JavaScript: For enhancing interactivity, allowing users to interact seamlessly with the platform, such as submitting reports and processing donations. C#: Used on the backend to handle data management, user authentication and server-side logic, ensuring a secure and efficient application.
